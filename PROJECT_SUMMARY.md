# AI 리터러시 챌린지 웹앱 개발 완료 보고서

## 📋 프로젝트 개요

**프로젝트명**: AI 리터러시 챌린지: 『진짜? 가짜?』  
**버전**: 2.0 (완성)  
**개발 기간**: 2025년 1월 12일  
**개발자**: AI Assistant (Claude) + 사용자 협업  
**기술 스택**: HTML5, CSS3, JavaScript (ES6+)  
**디자인**: Windows XP 스타일 컬러풀 디자인

## 🎯 프로젝트 목표

생성형 AI에 대한 학생들의 비판적 사고 및 기초 소양(Literacy) 함양을 위한 인터랙티브 웹앱 개발

### 주요 목표
- AI 생성 콘텐츠와 실제 콘텐츠를 구별하는 경험 제공
- AI 기술에 대한 균형 잡힌 시각 형성
- 비판적 사고 능력 향상
- 자기주도적 학습 태도 형성

## 🏗️ 구현된 기능

### 1. Screen 1.0: 시작 및 유형 선택 화면
- **메인 타이틀**: "진짜? 가짜?" 
- **부제목**: "AI가 만든 콘텐츠, 구별할 수 있나요?"
- **챌린지 선택**: 이미지 챌린지 / 텍스트 챌린지
- **반응형 디자인**: 데스크톱/모바일 최적화

### 2. Screen 2.0: 챌린지 진행 화면

#### 2.1 이미지 챌린지
- **Level 1 (튜토리얼)**: 명백한 AI 오류 (6개 손가락, 부자연스러운 배경)
- **Level 2 (훈련)**: 미묘한 오류 (빛과 그림자 불일치, 어색한 질감)
- **Level 3 (실전)**: 기술적으로 정교하지만 현실성 떨어지는 상황
- **힌트 기능**: Level 2부터 제공
- **즉시 피드백**: 정답/오답 즉시 표시 및 해설

#### 2.2 텍스트 챌린지 (진실 vs 거짓)
- **Level 1 (튜토리얼)**: 명백한 논리적 모순
- **Level 2 (훈련)**: 그럴듯하지만 논리적 비약 포함
- **Level 3 (실전)**: 맥락적 추론 능력 요구
- **주제**: 모든 텍스트가 'AI 리터러시' 관련 내용
- **힌트 기능**: Level 2부터 제공
- **판단 방식**: "진실" vs "거짓" 형태의 OX 퀴즈

### 2.5 Screen 2.5: 챌린지 결과 화면 (NEW!)
- **정답 개수 표시**: 원형 점수 표시기로 시각적 효과
- **점수별 메시지**: 80% 이상(훌륭), 60-79%(좋음), 40-59%(보통), 40% 미만(부족)
- **맞춤형 피드백**: 점수에 따른 격려 및 학습 유도 메시지
- **진행 흐름**: 챌린지 완료 → 결과 화면 → 성찰 질문

### 3. Screen 3.0: 성찰 질문 화면
- **진입 조건**: 3단계 챌린지 완료 후
- **질문 형태**: 3지 선다 객관식
- **챌린지별 맞춤 질문**: 이미지/텍스트 챌린지별 다른 질문
- **학습 효과**: 정답을 맞출 때까지 정답 숨김
- **진행 방식**: 3개 질문 모두 정답 시 다음 단계

### 4. Screen 4.0: 최종 결과 및 마무리 화면
- **완료 메시지**: 학습 목표 및 의미 설명
- **네비게이션**: 홈으로 / 다른 챌린지 선택
- **조건부 버튼**: 완료한 챌린지에 따라 다른 버튼 표시

## 🎨 디자인 특징

### Windows XP 스타일 적용 (v2.0 업데이트)
- **색상 팔레트**: 파란색 계열 그라데이션 기반
- **배경**: 파란색 그라데이션 (`#4a90e2` → `#7bb3f0` → `#a8d0f0`)
- **버튼 디자인**: 
  - AI 버튼: 노란색 그라데이션 (경고/주의)
  - 실제/진실 버튼: 파란색 그라데이션 (신뢰/안정)
  - 힌트 버튼: 주황색 그라데이션 (도움말)
  - 다음/계속 버튼: 파란색/초록색 그라데이션
- **그림자 효과**: `box-shadow`로 입체감 연출
- **폰트**: Tahoma, Noto Sans KR
- **애니메이션**: 호버 시 살짝 위로 올라가는 효과

### 반응형 디자인
- **모바일**: 360px 이상 지원
- **태블릿**: 768px 이상 지원  
- **데스크톱**: 1920px까지 지원
- **유연한 레이아웃**: Grid와 Flexbox 활용

### 시각적 피드백
- **정답**: 초록색 그라데이션 (`#10b981`)
- **오답**: 빨간색 그라데이션 (`#ef4444`)
- **비활성화**: 회색 그라데이션 (`#9ca3af`)
- **애니메이션**: 정답 시 펄스, 오답 시 흔들림 효과
