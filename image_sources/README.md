# 이미지 자산 디렉토리

이 디렉토리는 AI 리터러시 챌린지 웹앱에서 사용되는 이미지 자산들을 관리합니다.

## 디렉토리 구조

### `/ai_images/`
- AI가 생성한 이미지들을 저장합니다.
- Level 1: 명백한 오류가 있는 AI 이미지 (6개 손가락, 부자연스러운 배경 등)
- Level 2: 미묘한 오류가 있는 AI 이미지 (빛과 그림자 불일치, 어색한 질감 등)
- Level 3: 기술적으로 정교하지만 현실성이 떨어지는 AI 이미지

### `/real_images/`
- 실제 촬영된 사진들을 저장합니다.
- 각 레벨에 대응하는 실제 사진들
- 자연스러운 조명, 일관된 그림자, 정상적인 신체 부위 표현

### `/ui/`
- 앱의 아이콘, 로고 등 UI 관련 이미지들을 저장합니다.

## 이미지 파일 명명 규칙

- AI 이미지: `level{레벨}_{순번}.jpg` (예: level1_1.jpg, level2_1.jpg)
- 실제 이미지: `level{레벨}_{순번}.jpg` (예: level1_1.jpg, level2_1.jpg)
- UI 이미지: `{용도}_{이름}.{확장자}` (예: icon_logo.png, background_main.jpg)

## 주의사항

- 모든 이미지는 저작권이 없는 이미지이거나 적절한 라이선스를 가진 이미지를 사용해야 합니다.
- 이미지 크기는 웹 최적화를 위해 적절히 조정되어야 합니다.
- 이미지 로드 실패 시 대체 이미지가 표시됩니다.
